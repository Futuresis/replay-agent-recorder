# Changelog

All notable changes to Replay will be documented in this file.

This project follows a lightweight changelog format while it is in alpha.

## 0.1.0 - Alpha

- Initial alpha packaging for the Replay agent run recorder.
- Added recording, replay, forking, local tool recording, sandboxed filesystem
  capture, and trace visualization workflows.
- Added deterministic Agent4 demo coverage for local record/replay scenarios.
